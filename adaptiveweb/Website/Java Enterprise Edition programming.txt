Java EE Programming From Wikibooks, open books for an open world Jump to navigation Jump to search Wikipedia has related information at Java Platform, Enterprise Edition Java EE stands for Java 2 Platform, Enterprise Edition. This book will describe how to build Java EE based applications. The implementation is done with the help of XDoclet which simplifies the process of building an application. Aim[edit] How to design and develop various Java EE components.
